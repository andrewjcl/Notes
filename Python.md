## Structure
*flow*
```python
if numberX < 5:                     # if statement
  print('NumberX is below 5')
# resume flow  
 
  
while numberX < 5:                  # while statement
  print('NumberX is below 5')
  numberX = numberX + 1
# resume flow
```




*Operators*
```python
2 ** 3    # exponent            2 ** 3 = 8
22 // 8   # floored quotient   22 // 8 = 2
22 % 8    # modulus             22 % 8 = 6
```
